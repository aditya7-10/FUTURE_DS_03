📌 Project Overview

This project was developed as part of the Future Interns Data Science & Analytics Internship (Task 3). The objective was to analyze customer behavior throughout the marketing funnel and evaluate conversion performance using an interactive Power BI dashboard.
The dashboard tracks how users progress from Website Visitors → Product Views → Add to Cart → Purchases, helping identify customer drop-off points, measure marketing effectiveness, and generate actionable business insights. It also compares performance across marketing channels, campaign types, devices, and regions to support data-driven decision-making.

🎯 Objectives
Analyze customer journey through the marketing funnel
Identify major drop-off stages
Measure conversion performance
Compare marketing channel effectiveness
Evaluate campaign performance
Track revenue trends
Build an interactive dashboard for business insights
